# Mi'kmaq Game

Welcome to the **Mi'kmaq Game** repository! This project is an engaging and educational game inspired by the rich culture and traditions of the Mi'kmaq people. The game is designed to be interactive, fun, and informative, bringing cultural elements to life through modern technology.

## Features

- **Interactive Gameplay**: Experience an engaging game built using modern web technologies.
- **Cultural Representation**: Learn about the Mi'kmaq culture and traditions through immersive storytelling and gameplay.
- **Responsive Design**: Optimized for devices of all sizes, ensuring a seamless experience on both desktop and mobile platforms.

## Technology Stack

This project is built using the following technologies:

- **JavaScript** (55.5%): Core functionality and interactivity.
- **HTML** (33.5%): Structure and content of the game.
- **CSS** (11%): Styling and responsive design.
